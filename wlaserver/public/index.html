About The App

WildlifeAlerts is a mobile app that provide the general public with the option to post an alert regarding an animal in need. This triggers a notification sent to wildlife rescuers in that region who register through the app. Key Points:

  • Anyone can download and use the app to post an alert about an animal in need.
  • Alerts contain user-generated text, optional images, and location data, providing details about the situation
  • This information is only able to be viewed by authenticated wildlife rescuers that register through the app.

Data Security & Privacy

To register as a wildlfe rescuer, social sign up is optional, and the app uses this for authentication purposes and to perform the main funciton of the app which is to deliver notifications to specific users based on an identifier such as email, and name. The acquired data solely facilitates the app's main functions and is not repurposed or shared with third parties.
